Microphone Blocked

Browser security blocks the microphone on local network URLs (like http://192.168.x.x:3000) unless they are secure.

How to bypass in Chrome (Mobile/PC):

  1. Open Chrome and go to:
    chrome://flags/#unsafely-treat-insecure-origin-as-secure
  2. Toggle that flag to Enabled.
  3. In the text area, enter this URL:
    http://192.168.1.100:3000
  4. Relaunch Chrome. The microphone and camera will work perfectly!
